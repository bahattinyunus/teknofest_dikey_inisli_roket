# Contributing to Vertical-Arch

Thank you for your interest in contributing to the Vertical Landing Rocket System Architecture project!

## How to Contribute

1.  **Fork the Repository:** Create your own copy of the project.
2.  **Create a Branch:** `git checkout -b feature/your-feature-name`.
3.  **Implement Changes:** Ensure your code or documentation follows the project's standards.
4.  **Commit Changes:** `git commit -m "Add some feature"`.
5.  **Push to Branch:** `git push origin feature/your-feature-name`.
6.  **Open a Pull Request:** Describe your changes and why they are beneficial.

## Technical Standards
- High-reliability focus for all control algorithms.
- Clear documentation for all hardware interfaces.
- Deterministic timing for RTOS-based software components.
